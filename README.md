# Pokemon Battle Web App

This is a simple turn-based Pokemon battle game built with React, Vite, and Tailwind CSS.

## Features
- **Random Matchups**: Every game fetches two random Gen 1 Pokemon from the PokeAPI.
- **Battle System**: Turn-based combat with simplified damage calculation based on stats.
- **Responsive UI**: Modern, card-based interface with health bars and animations.

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`

## Tech Stack
- React
- Tailwind CSS
- PokeAPI
- Vite
